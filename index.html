Wichtiger Hinweis zu Dateinamen mit Leerzeichen:
Obwohl moderne Browser und GitHub Pages mit Leerzeichen in Dateinamen umgehen können (sie werden im Link durch %20 ersetzt), ist es oft eine bessere Praxis, Leerzeichen durch Bindestriche (-) oder Unterstriche (_) zu ersetzen (z.B. Umrechner-Pro.html). Dies macht die Links manchmal lesbarer und vermeidet potenzielle Probleme mit manchen Systemen. Wenn du die Namen so lässt, funktionieren die obigen Links, aber der tatsächliche Link im Browser wird z.B. .../Umrechner%20Pro.html lauten.
